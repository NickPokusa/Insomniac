Instructions for Game Play
1. Get the Game - Navigate to https://github.com/NickPokusa/Insomniac
2. Download the zip file: Insomniac Release 1.0
3. Open the zip file and select the application. This will prompt you to extract all files. Click Extract all.
4. Open the new folder and double-click the application file
5. Double-click the file to launch the game
6. The game will launch automatically into a new window

Controls
1. Click your mouse to interact with objects
2. Click the SLEEP button at the bottom of the screen to increase the Sleep Meter
3. Sleep meter stops when a threat comes
4. Click the left and right arrows to find the threat
5. Click the environment object to respond to the threat with light.
6. When the threat disappears, restart the sleep meter
7. Continue searching and responding to threats
8. The Night is complete when the sleep meter reaches 100%
9. Continue to play until all Nights are completed
